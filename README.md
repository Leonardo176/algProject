# Algorithm and Data Structures Project

## Project Goal

The goal of this project is to implement the following data structures in the Python:

- *Binary Search Tree* (BST)
- *Adelson-Velsky and Landis Tree* (AVL)
- *Red-Black Tree* (RBT)

After implementing these data structures, a time complexity analysis will be
included as an integral part of the final paper.

## Course Context

The project is being developed as part of the *Algorithms and Data Structures*
course at the *University of Udine* for the Academic Year 2025-2026.
